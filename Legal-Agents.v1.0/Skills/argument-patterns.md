# ARGUMENT PATTERNS

Estruturas argumentativas e técnicas para construção de teses jurídicas.

---

## IRAC+ (Estrutura Base)

### Componentes

**I - Issue (Questão)**
- Reformular a questão jurídica com precisão técnica
- Delimitar exatamente o que se discute
- Uma questão por vez

**R - Rule (Regra)**
- Normas aplicáveis em ordem hierárquica
- Precedentes vinculantes primeiro
- Jurisprudência dominante
- Doutrina (se necessário)

**A - Application (Aplicação/Subsunção)**
- Para cada elemento da norma, identificar o fato correspondente
- Demonstrar a adequação (ou inadequação)
- Explicitar escolhas interpretativas

**C - Counterarguments (Contrapontos)**
- Antecipar argumentos contrários
- Refutar ou distinguir
- Não ignorar pontos fracos

**+ - Conclusion (Conclusão)**
- Enunciar a tese
- Qualificar confiança
- Listar condições e ressalvas

## Padrões Hermenêuticos

### 1. Interpretação Literal
**Quando usar**: Norma clara, sem ambiguidade
**Como aplicar**:
```
Norma diz X → Fato é X → Consequência é Y
```
**Limitação**: "In claris cessat interpretatio" é mito; sempre há interpretação

### 2. Interpretação Sistemática
**Quando usar**: Norma em contexto de outras normas
**Como aplicar**:
- Verificar normas relacionadas
- Buscar coerência com o sistema
- Aplicar princípios gerais do microssistema
```
Norma A + Norma B + Princípio C → Interpretação D
```

### 3. Interpretação Teleológica
**Quando usar**: Conceitos indeterminados, cláusulas gerais
**Como aplicar**:
- Identificar finalidade da norma
- Verificar exposição de motivos (se disponível)
- Aplicar para realizar a finalidade
```
Norma visa proteger X → Interpretação que melhor protege X
```

### 4. Proporcionalidade (Alexy)
**Quando usar**: Colisão de princípios, direitos fundamentais
**Como aplicar**:
1. **Adequação**: Medida é apta a alcançar o fim?
2. **Necessidade**: Há medida menos gravosa?
3. **Proporcionalidade estrita**: Benefício justifica o ônus?

### 5. Integração (Lacuna)
**Quando usar**: Ausência de norma específica
**Como aplicar** (art. 4º LINDB):
1. Analogia legis (norma similar)
2. Analogia iuris (princípios do sistema)
3. Princípios gerais do direito
4. Equidade (se autorizada)

### 6. Resolução de Antinomias
**Quando usar**: Conflito entre normas
**Critérios** (em ordem):
1. **Hierárquico**: Superior prevalece
2. **Especialidade**: Especial prevalece sobre geral
3. **Cronológico**: Posterior prevalece

## Técnicas Argumentativas

### Argumento a fortiori
"Com maior razão"
- **A maiori ad minus**: Se pode o mais, pode o menos
- **A minori ad maius**: Se não pode o menos, não pode o mais

```
Se X que é mais grave tem consequência Y,
então Z que é menos grave também tem consequência Y (ou menor)
```

### Argumento a contrario sensu
"Em sentido contrário"
```
Se a norma diz que A implica B,
então não-A implica não-B
```
**Cuidado**: Nem sempre válido; verificar se norma é taxativa

### Argumento ab absurdo
Demonstrar que interpretação contrária leva a resultado absurdo
```
Se interpretarmos assim, a consequência seria X
X é absurdo/impossível/contrário ao sistema
Logo, a interpretação correta é outra
```

### Argumento de autoridade
Invocar posição consolidada
```
O STJ, em reiterados julgados, entende que...
A doutrina majoritária, representada por [autor], sustenta que...
```
**Hierarquia**: Precedente vinculante > Jurisprudência dominante > Doutrina

### Distinguishing
Afastar precedente aparentemente aplicável
```
O precedente X não se aplica porque:
1. O fato determinante em X era A
2. No caso presente, o fato é B
3. Essa diferença é juridicamente relevante porque...
```

### Overruling (argumentar pela superação)
Demonstrar que precedente deve ser abandonado
```
O precedente X deve ser superado porque:
1. Fundamentos originais não mais subsistem
2. Houve alteração legislativa/social/valorativa
3. A manutenção causa injustiça sistemática
```
**Atenção**: Raramente acolhido; usar com cautela

## Estruturas de Tese

### Tese Principal
- Argumento mais forte
- Fundamentação mais sólida
- Jurisprudência favorável

### Tese Subsidiária
- "Ainda que não seja acolhida a tese principal..."
- Argumento alternativo
- Menor benefício, maior chance de êxito

### Tese Eventual
- "Ad argumentandum tantum..."
- Apenas para não precluir
- Sem expectativa real de acolhimento

### Pedidos Cumulados
| Tipo | Relação | Exemplo |
|------|---------|---------|
| Simples | Todos independentes | Danos + honorários |
| Subsidiário | Se não A, então B | Rescisão ou revisão |
| Alternativo | A ou B (autor escolhe) | Substituição ou abatimento |
| Sucessivo | Se A, então B | Nulidade, então restituição |

## Fundamentação de Pedidos

### Pedido Condenatório
```
Elementos obrigatórios:
1. Conduta ilícita (ação ou omissão)
2. Dano (material ou moral)
3. Nexo causal
4. Quantificação ou critérios
```

### Pedido Declaratório
```
Elementos obrigatórios:
1. Incerteza jurídica objetiva
2. Interesse de agir
3. Relação jurídica (não mero fato)
```

### Pedido Constitutivo
```
Elementos obrigatórios:
1. Fato jurídico que autoriza modificação
2. Legitimidade
3. Requisitos legais da constituição/desconstituição
```

### Tutela de Urgência (CPC art. 300)
```
Requisitos:
1. Probabilidade do direito (fumus boni iuris)
2. Perigo de dano ou risco ao resultado útil (periculum in mora)
3. Reversibilidade (salvo exceções)
```

## Vícios a Evitar

### Na Subsunção
- Salto lógico (fato não demonstra elemento)
- Generalização indevida
- Petição de princípio (presumir o que deve provar)

### Na Fundamentação
- Citar norma sem demonstrar aplicação
- Invocar jurisprudência sem pertinência
- Ignorar posição contrária dominante

### Na Estrutura
- Argumentos contraditórios sem ressalva
- Pedido sem fundamento correspondente
- Conclusão que não decorre das premissas
