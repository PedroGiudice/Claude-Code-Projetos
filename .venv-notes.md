# Virtual Environment Notes

## Project Root venv

Created: 2025-11-15

**Purpose:** Satisfy RULE_006 (venv obrigatório) at project root level.

This venv is primarily for:
- Running project-level scripts
- Satisfying venv-check hook requirement
- Development tools and linters

Individual agents have their own venvs in `agentes/*/venv/`.

## Activation

### Opção 1: Manual (para terminal/scripts)
```bash
source .venv/bin/activate
```

### Opção 2: Iniciar Claude Code com venv (RECOMENDADO para RULE_006)
```bash
./start-claude-with-venv.sh
```

Este script:
- Ativa o venv automaticamente
- Exporta VIRTUAL_ENV para todos os hooks
- Elimina o warning de RULE_006 na statusline
- Inicia Claude Code com ambiente correto

## Dependencies

Currently minimal - install as needed:

```bash
pip install --upgrade pip

```
